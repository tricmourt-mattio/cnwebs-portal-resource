# cnwebs-portal-resource

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面，作为网页资源的集中存储库。仓库内容不针对任何特定域名或网站，仅用于存放通用型网页文件及相关资源。

## 目录结构

```
cnwebs-portal-resource/
├── pages/           # 存放独立 HTML 页面文件
├── assets/          # 页面依赖的静态资源（CSS、JS、图片等）
├── archive/         # 历史版本或已归档的文件
└── README.md        # 本文件
```

- **pages/**：包含可直接访问的 HTML 页面，每个页面为一个独立功能模块。
- **assets/**：存放页面引用的样式、脚本、字体、图标等资源文件。
- **archive/**：用于存放不再活跃更新但需保留的页面或资源。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面之间相互独立，无跨页面依赖关系。
- 归档时请保持文件结构清晰，避免冗余或冲突的命名。
- 建议为新页面创建独立子目录，方便管理。

## 维护说明

- 本仓库由维护者不定期更新，不接受外部直接推送修改。
- 如需提交页面或资源，请先通过 Issues 或内部沟通渠道联系维护者。
- 所有文件应符合通用网页规范，避免包含恶意代码或敏感内容。
- 仓库内容仅作归档用途，不提供任何形式的服务可用性保证。

## 使用方式

可直接通过 GitHub Pages 或其他静态托管服务访问 pages/ 目录下的 HTML 文件。具体访问路径请参考仓库部署配置。
